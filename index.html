git config --global user.name blabla - первинні налаштування, додавання ім'я яке буде відображатись при комітах git config --global user.email blabal@bla - первинні налаштування, додавання емейлу який буде відображатись в інформації про виконані вами коміти git init - ініціалізація локального репозиторію git status - команда для перевірки готовності репизиторія для коміта git add - (індексуе) додати файл в коміт git commit - фіксування проіндексованих файлів та зберігання їх в історію git log - інформація про користувача, та історію його дій git checkout - зміна робочої гілки git branch - створення нової гілки git push - відправити данні у віддаленний репозиторій git pull - стягнути данні з віддаленого репизиторію та об'єднати з локальним git merge - внести зміни з поточної гілки в основну git clone -скачати репозиторій з історією всіх змін .gitignore нужен для скрытия файлов и папок от системы контроля версий Git. Обычно скрывают конфигурационные файлы (особенно с паролями), временные файли и папки. gitignore использует glob формат для выборки файлов.
